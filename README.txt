1) run.sh to start the game

Also available:

dunk_right.py  - fires the horn and right feathers & water
dunk_left.py   - fires the horn and left feathers & water
dunk_both.py   - fires the horn and both feathers & water

2) initial startup takes 20-30 seconds BEFORE you see any output

Therefore the game will display a prompt and re-run without exiting.

game will display a y/n prompt

     Press control-c to exit game

     Press y to start the game (n just repeats the prompt)

Currently the game is set to:

120 seconds for the game
10 second polling delay
